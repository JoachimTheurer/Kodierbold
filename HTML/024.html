In p5.js, das auf JavaScript basiert, stehen dir verschiedene Kontrollstrukturen zur Verfügung, um den Ablauf deines Programms zu steuern. Hier sind die wichtigsten: Bedingte Anweisungen: Ermöglichen es, Code nur unter bestimmten Bedingungen auszuführen.
if-Anweisung: Führt Code aus, wenn eine Bedingung wahr ist. javascript Kopieren Bearbeiten if (x > 10) { // Code wird ausgeführt, wenn x größer als 10 ist }
if...else-Anweisung: Führt Code aus, wenn die Bedingung wahr ist, andernfalls wird ein anderer Codeblock ausgeführt. javascript Kopieren Bearbeiten if (x > 10) { // Code für den Fall, dass x größer als 10 ist } else { // Code für den Fall, dass x nicht größer als 10 ist }
else if-Anweisung: Ermöglicht die Prüfung zusätzlicher Bedingungen, wenn die vorherige Bedingung falsch war. javascript Kopieren Bearbeiten if (x > 10) { // Code für x > 10 } else if (x > 5) { // Code für x > 5 und x <= 10 } else { // Code für x <= 5 }

Schleifen: Wiederholen Codeblöcke, solange eine bestimmte Bedingung erfüllt ist.


for-Schleife: Wiederholt Code eine festgelegte Anzahl von Malen. javascript Kopieren Bearbeiten for (let i = 0; i < 5; i++) { // Code wird 5-mal ausgeführt }
while-Schleife: Führt Code aus, solange eine Bedingung wahr ist. javascript Kopieren Bearbeiten while (x < 10) { // Code wird ausgeführt, solange x kleiner als 10 ist }
do...while-Schleife: Führt Code mindestens einmal aus und wiederholt ihn, solange die Bedingung wahr ist. javascript Kopieren Bearbeiten do { // Code wird mindestens einmal ausgeführt } while (x < 10);

Switch-Anweisung: Ermöglicht die Auswahl zwischen verschiedenen Codeblöcken basierend auf dem Wert einer Variablen oder eines Ausdrucks.

javascript Kopieren Bearbeiten switch (expression) { case value1: // Code für den Fall, dass expression gleich value1 ist break; case value2: // Code für den Fall, dass expression gleich value2 ist break; default: // Code für alle anderen Fälle } Diese Kontrollstrukturen helfen dir, den Ablauf deines Programms flexibel und dynamisch zu gestalten.
